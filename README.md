# joe_notes_grabber
Grab joe's notes
Ha ha
```sh
usage: joe_notes_grabber.py [-h] --url url [-u username] [-p password]
                            [-e [exts [exts ...]]]

Hyperlink grabber

optional arguments:
  -h, --help            show this help message and exit
  --url url             Webpage to grab from
  -u username, --user username
                        Username for basic auth
  -p password, --password password
                        Password for basic auth
  -e [exts [exts ...]], --extension [exts [exts ...]]
                        File extension to grab. Default: .pdf .ppt .doc .docx
```
